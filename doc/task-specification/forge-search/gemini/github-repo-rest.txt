Окей, вот основные параметры, которые вы можете использовать для поиска репозиториев через GitHub REST API (конечная точка GET /search/repositories). Эти параметры передаются как query parameters в URL.Полную и самую актуальную документацию всегда лучше смотреть на официальном сайте: GitHub REST API Docs - Search RepositoriesОсновной параметр запроса:•q (обязательный, string): Сам поисковый запрос. Он может содержать:•Ключевые слова для поиска в названии, описании и README репозитория.•Квалификаторы (qualifiers) для уточнения поиска по различным атрибутам.Квалификаторы (Qualifiers) для параметра
