# SourceChew

An open-source multiplatform (Android, iOS, Desktop, Web) application for convenient reading of source code from repositories on GitHub.

It allows you to search and navigate through repositories, users, and organizations, using a tree-based structure, with support for offline reading.

---

## 🚀 Features

* 🔍 Search repositories, users, and organizations (in progress)
* 🌲 Tree-based navigation of repository structure (planned)
* 📄 Read source code (planned: syntax highlighting)
* 📦 Offline support (planned)
* 🌐 Multiplatform: Android, iOS, Desktop, Web
